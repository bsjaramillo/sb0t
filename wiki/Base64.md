# +Base64+


+**decode**+

**Type:** Function
**Return:** string
**Description:** convert a base64 encoded string back to text

```javascript
var str = "hello";
var encoded = Base64.encode(str);
var decoded = Base64.decode(encoded);
```

+**encode**+

**Type:** Function
**Return:** string
**Description:** convert text to a base64 encoded string

```javascript
var str = "hello";
var encoded = Base64.encode(str);
```
